# System rzymski

- $V = 5$
- $X = 10$
- $L = 50$
- $C = 100$
- $D = 500$
- $M = 1000$

W klasycznym systemie rzymskim nie stosowano zasady "odejmowania". Liczba $4$ wyrażana była przez $IIII$, a nie przez $IV$. Dzięki temu był on czystym systemem znakowym, w którym kolejność zapisu cyfr nie miała znaczenia dla finalnej wartości, choć przyjęto że znaki zapisywano w kolejności malejącej.
